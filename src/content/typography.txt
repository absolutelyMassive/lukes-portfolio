TYPOGRAPHY — BASIC SYSTEM
=========================

FONT
----
Family: Satoshi (self-hosted, weight 500 / medium).
Fallbacks: ui-sans-serif, system-ui, sans-serif.

All UI and content use the same weight (500) for a single, calm rhythm.


SCALE (use with existing color tokens)
--------------------------------------
overline   — uppercase, wide tracking, xs, secondary color. Eyebrows / labels.
caption    — xs, secondary. Meta, timestamps, footnotes.
body       — sm, secondary for long text; primary for emphasis in UI.
title      — xl. Section titles.
display    — 3xl, tight leading. Page hero / major headings.


LINE HEIGHT
-----------
Tight on display and large headings. Relaxed on body when paragraphs run long.


RULES OF THUMB
--------------
- One display-level line per view where possible.
- Prefer body + caption for dense UI; avoid shrinking below caption.
- Narrow columns still work well at a single medium weight.
