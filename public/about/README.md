# About floating photos

Six **200×200px** (1:1) images shown behind the center about card when `?about=1`.

| File | Figma placement |
|------|-----------------|
| `float-01.png` | Top left |
| `float-02.png` | Bottom left (may clip off-screen) |
| `float-03.png` | Mid left |
| `float-04.png` | Bottom center-right |
| `float-05.png` | Mid right |
| `float-06.png` | Top right |

Replace any file to update a photo. Paths and layout are configured in `src/content/aboutFloatingImages.ts`.
