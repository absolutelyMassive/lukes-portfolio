# Fonts

Self-hosted files served at `/fonts/…`.

| File | Use |
|------|-----|
| `PPNeueMachina-Regular.otf` | About overlay (`?about=1`), gallery (`/gallery`) |

To swap the file, replace the `.otf` here and keep the same filename. The `@font-face` rule lives in `src/fonts/pp-neue-machina.css` (imported from `layout.tsx`).
